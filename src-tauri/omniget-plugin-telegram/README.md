# OmniGet Telegram Plugin

Download media from Telegram channels and groups.

## Features

- QR code login (scan from Telegram app)
- Phone number + verification code login
- Two-factor authentication support
- Browse channels and groups
- Download photos, videos, files, and audio
- Batch download with progress tracking
- Thumbnail cache for fast browsing
- Search within chats

## Install

This plugin is bundled with OmniGet. It can also be installed from the in-app Marketplace.

## Development

```bash
cargo check -p omniget-plugin-telegram
```
